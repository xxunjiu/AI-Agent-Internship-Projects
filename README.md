# 🌍 4. 智能旅行规划助手（Trip Planner Agent）

## 🎯 项目说明
基于 LangGraph 的多智能体旅行规划系统，可自动生成完整旅行行程。

---

## 🧠 系统能力

- 📍 景点智能推荐（高德地图 API）
- 🌤 天气信息查询
- 🏨 酒店推荐系统
- 🧭 多 Agent 协作规划
- 📦 结构化 JSON 输出

---

## 🏗 系统架构

用户输入
→ 景点 Agent（高德 API）
→ 天气 Agent
→ 酒店 Agent
→ Planner Agent（整合生成）
→ 输出完整旅行计划

---

## ⚙️ 技术栈
Python / LangGraph / LLM API / Streamlit / 高德地图 API

---

## 🚀 运行方式
python trip_planner_agent.py

---

## ⭐ 项目亮点

- 多智能体协作系统（Multi-Agent Architecture）
- 真实 API 调用（高德地图）
- 自动旅行规划生成
- 可扩展为智能导游系统
- 结构化 JSON 输出
